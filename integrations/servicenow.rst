ServiceNow Integration
=======================

Overview
^^^^^^^^
Scalr provides out-of-the-box integration with ServiceNow to enable mutual customers to continue to leverage ServiceNow for existing ITSM workflows including problem management, incident management, change management, asset management, etc. The Scalr ServiceNow integration allows users to maintain a single source of truth for all on premise and cloud-based infrastructure resources and services,  and leverage an existing workflows, while taking advantage of Scalr for cloud infrastructure management.
